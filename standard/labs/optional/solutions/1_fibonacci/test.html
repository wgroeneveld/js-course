Secret Extra Lab: Fibonacci

Given a Fibonacci row: 0, 1, 1, 2, 3, 5, 8, 13, 21, ...

Implement this simple algorithm in Javascript. Make sure the complexity isn't more than n! For the unit tests to work, you'll need to keep track of the fibonacci # runs on window scope - see QUnit source code.

Public interface: 
fibonacci(n); // implement me
fiboCounter;  // get number of fibonacci() runs

Handy tips:

  1. Start with the simplest thing possible (the algorithm, without worrying about complexity) Make sure some unit tests are green.
  2. Use Closures to keep track of what you've already calculated! You should not write this the "procedural" way.
    You'll only need 2 functions on window scope.
  3. Combine the previous tip with this one: the Decorator pattern.
  4. Don't use typeof() but hasOwnProperty()!
