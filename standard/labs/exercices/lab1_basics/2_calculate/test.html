Lab 1: Calculations!

Write a function which evaluates a simple formula in strinform and returns the calculated the result.
It has to be possible to execute both plus and multiply (x en *) formulas.
You do not need to take special cases into account (empty strings, malformed formulas, ...).
Your function will only get string primitives as arguments and will only return number primitives as results.
More examples can be found inside the unit tests! One practical Example: 
calculate("3 + 4") === 7;
calculate("10 * 10") === calculate("10 x 10") === 100;
calculate("455") === 455;
This'll help you on your way: JS String API.
