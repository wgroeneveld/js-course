Lab 3: filtering on age

We've got an array of persnos with different ages. We would like to easily retrieve people who are younger or older than a certain age. Write the functions isYoungerThan en isOlderThan which'll act as your filter function.
Don't loop over the array yourself! (You're doing it wrong) See examples below: 

  var jan = {name: 'jan', age: 20};
  var piet = {name: 'piet', age: 30};
  var joris = {name: 'joris', age: 40};
  var korneel = {name: 'korneel', age: 50};
  var people = [jan,piet,joris,korneel];

  people.filter(isYoungerThan(35)); // -> [jan, piet]
  people.filter(isOlderThan(35)); // -> [joris, korneel]
