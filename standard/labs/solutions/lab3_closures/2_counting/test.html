Lab 3: creating counter functions

We want to be able to easily create a function that counts up. Create a function "createCounter" that returns a function. Every time we call the returned function, it will count up by 1. We should be able to create multiple independent counter functions.
See examples below: 

  var counterFunction1 = createCounter();
  var counterFunction2 = createCounter();
  counterFunction1(); // returns 1
  counterFunction1(); // returns 2
  counterFunction2(); // returns 1
