Lab 5: Dactylo game revisited

This is the continuation of the dactylo lab you've done in series 4.

In the previous exercice, you developed a simple dactylo game that only checks one character. In this lab, you'll need to be able to typee/check a stream of text. 

    var game = DACTYLO.createGame({expectedCharacters:'abc'});
    game.start();
    // wait 3 seconds
    game.type('b');
    game.type('c');
    game.amountOfPenalties; // === 1

A second requirement is to be able to add 'listeners' to the dactylo game. These functions will get called every time a penalty has been assigned. 

    var game = DACTYLO.createGame({expectedCharacters:'ab'});
    game.addListener(function () { console.log('you\'re doing it wrong!');});
    game.start();
    game.type('a');
    game.type('a'); // "you're doing it wrong!" output in console
