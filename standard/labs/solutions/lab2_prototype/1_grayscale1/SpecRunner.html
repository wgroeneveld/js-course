Lab 2: Grayscale exercice

Make a function which can create pixel objects, and can convert those to their grayscale values.
Given:

  • A pixel consisting of 4 values: RGBA
  • The gray value of a pixel is the average of the RGB values. The alpha values are UNaltered.

This is the public interface: 

var pixels = {
    create: function(red, green, blue, alpha) {
      // returns a new pixel
  }
};

var pixel = {
  toGrayscale: function() {
    // returns a new pixel object with grayscale values
  }
}
So I should be able to do something like this: 
var pixel1 = pixels.create(10, 20, 30, 5);
pixel1.red === 10;
pixel1.green === 20;
pixel1.blue === 30;
pixel1.alpha === 5;
var grayscalePixel = pixel1.toGrayscale();  // === pixels.create(20, 20, 20, 5);
