Lab 2: Gardener exercice

Write a program that can sow and plant seeds. Seeds create beautiful flowers, who recreate seeds by growing.
Every seed that gets sown has to reproduce a new plant.

Growing a plant creates new seeds that are based upon the seed which created that flower.
Seeds keep track of generations of ancestors, starting with 1.

This is the public interface: 

var seed = {
    sow: function() {
      // returns plant type for this
  }
};

var plant = {

  grow: function() {
     // grow a little: return a new generation seed of this plant, based upon the previous one!
  }
};
I should be able to do this: (Take a look at the unit test source code!!) 
var pumpkinSeed = Object.create(seed);
var pumpkin = pumpkinSeed.sow();

var newPumpkinSeed = pumpkin.grow();
newPumpkinSeed.generation === 2;

var newerPumpkinSeed = newPumpkinSeed.sow().grow();
newerPumpkinSeed.generation === 3;

Draw us a nice schematic on how the third genration seed's prototype looks like!
What happens when I call the sow() function no that third generation seed? Draw the necessary connections!
