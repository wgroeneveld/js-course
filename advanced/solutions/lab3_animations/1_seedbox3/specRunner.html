Lab 1: Gardening in the Seedbox part III

It's time to add animations to sowing seeds and growing plants. Start by refactoring your code into a public module which exposes the following functions: 

var gardening = {
	buySeeds: function(amount) {
		// places x seeds onto the seedbox
	},
	mowLawn: function(spots) {
		// initializes the spots on the lawn
	},
	cleanup: function() {
		// should throw away all seeds and clear spots on the lawn.
	}
};

In addition to doing that, we now add some animations:

  • When sowing a seed, it should move towards the first available spot on the lawn.
  • When a seed has been placed onto the lawn, it should slowly be replaced by the plant it has become.

When you want to animate the position of a seed, make sure the css attribute position is absolute, and the seed has been added as a direct child of the body. You can use detach() and appendTo() to move a seed element from the seedbox container to the body element.
