Lab 1: Twitter client - the tweets

You'll be making a twitter wall feed client in Javascript, isn't that awesome? Probably.
The first step is generating tweet message divs you can append to the twitterContainer div provided below.

The function will take in an array of objects, containing tweet data, something like this: 
generateTweets([{
		from_user: "Jos",
		profile_image_url: "bla.png",
		created_at: "10/12/2010",
		text: "Howdy all, I'm new at Javascript but I think it's a highly respectable language!"
	}, {
		from_user: "Urbain",
		profile_image_url: "cool.png",
		created_at: "12/03/1999",
		text: "I think JS Sucks, it's not statically typed and I want autocompletion! *Whine*"	
	},
	...
]);
Create a tweet div (with 'tweet' as class) and add an image as avatar, some text, and another div classed with 'timestamp'.
The end result will be something like: 
<div class="tweet">
	<img src="(profile_image_url)">
	(from_user) says (text)
	<div class="timestamp">(created_at)</div>
</div>
