Lab 1: Shoot a cow

(No cows were harmed during the making of this exercice)

Your assignment: Create a game module with two public methods start and stop. The start method should immediately add a div with id cow to the document. Add the cow to the body, there is no container div in this excercise. After 1 second the cow should disapear. It should reappear 1 second later at another (random) location.
Tips

  • Use setInterval or setTimeout
  • The global Math object has methods for generating random numbers, rounding them,..
