Lab 1: Twitter client part III - rotating the tweets on the wall

Take out your twitter wall solution from lab 2. Again, it's time to rewrite it into a module with this interface: 

var twitter = {
	addTweets: function(tweetArray) { ... }
};
Nothing more than that. The function addTweets now does more than merely displaying them in the twitterContainer:
  • Tweets will be rotating: if more than 3 tweets are visible, the oldest ones will slowly fade away.
  • There can never be more than 3 tweets displayed.
  • So: calling addTweets twice should rotate old tweets out and display the new ones.
You can safely assume the tweetArray will never contain more than 3 elements.
Don't forget to remove unneeded elements from the DOM tree using .remove()!
