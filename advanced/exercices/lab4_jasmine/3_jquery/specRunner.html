Lab 4: jQuery and Jasmine combined

Implement this test-first!
I want to grow a flower inside a pot. I start out with a new flowerpot and I'm able to grow the plant by watering it. Let's assume it came with a tiny seed pre-filled.
After watering, it changes into a beautiful flower. You'll need to implement something like this: 

var flowerpot = (function() {
	function water() { ... }
}); // notice I don't want to evaluate this yet, as it might add stuff into the DOM!
When you've created a flowerpot, a corresponding div with class 'flowerpot' appears into the DOM tree (doesn't matter where).
Users can water the pot by clicking on it. Be sure to test these events!
